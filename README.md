# Password-manger-and-genrator
Password Generator
A simple command-line password generator with entropy calculation to measure password strength.
Features

Generate Random Passwords - Create secure passwords with custom length and character types
Entropy Calculation - Measure password strength in bits
Password Strength Indicator - Get feedback on password security level (Weak, Fair, Good, Very Strong)
Save & View Passwords - Store generated passwords in a local file




Generate Password [1] - Create a new password

Enter desired length (8-16 characters)
Choose character types: low (lowercase), up (uppercase), digits, punc (punctuation)
Example input: low up digits


View Passwords [2] - Display all saved passwords from password.txt
Exit [3] - Close the program
